--------------------------------------------------------- STEP 2: OPEN TERMINAL AND INITIALIZE GIT --------------------------------------------------------- git init --------------------------------------------------------- This command creates a new Git repository. STEP 3: ADD FILES TO GIT --------------------------------------------------------- git add . --------------------------------------------------------- This command adds all files to the staging area. STEP 4: COMMIT THE FILES --------------------------------------------------------- git commit -m "My first portfolio website" --------------------------------------------------------- This command saves the changes. STEP 5: CREATE GITHUB REPOSITORY --------------------------------------------------------- 1. Go to www.github.com 2. Click on New Repository 3. Enter repository name as "portfolio" 4. Click Create Repository --------------------------------------------------------- STEP 6: CONNECT LOCAL PROJECT TO GITHUB --------------------------------------------------------- git branch -M main git remote add origin https://github.com/your-username/portfolio.git --------------------------------------------------------- STEP 7: PUSH CODE TO GITHUB --------------------------------------------------------- git push -u origin main --------------------------------------------------------- Now the project is uploaded to GitHub. STEP 8: DEPLOY USING GITHUB PAGES --------------------------------------------------------- 1. Open GitHub repository 2. Go to Settings 3. Click Pages 4. Select Branch as main 5. Click Save --------------------------------------------------------- Website URL: https://your-username.github.io/portfolio/ STEP 9: DEPLOY USING NETLIFY (OPTIONAL) --------------------------------------------------------- 1. Go to www.netlify.com 2. Login or Sign up 3. Click Add New Site 4. Drag and drop project folder --------------------------------------------------------- RESULT: --------------------------------------------------------- The portfolio website is successfully created, pushed to GitHub and deployed using GitHub Pages / Netlify. ---------------------------------------------------------
